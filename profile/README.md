## 👋 Hi there!

We're **Evergreen IT** — a software company from Den Haag with a soft spot for solving problems differently.

We build for enterprise on **OutSystems**, we build for fun on whatever tech got us curious that week, and we treat AI as a co-pilot rather than a gimmick. Sometimes the result is a serious low-code platform; sometimes it's a weekend MCP experiment that turns into something real.

Either way, we build it like it's meant to last.

## 🧠 What we believe

> *"It's always easy to do something the way it's always been done.*
> *We'd rather do it the way it hasn't been done yet."*

- 🤖 **AI-first** — not because we have to, but because we can
- 🎨 **Creative by default** — solutions over limitations
- 🌱 **Evergreen** — what we build keeps growing, long after we're done
- 🧪 **Playful** — if building isn't fun, we're doing something wrong
