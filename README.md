## 👋 Hi there!

We're **Evergreen IT** — a software company from the Netherlands that prefers to skip the well-worn path.
We build products where AI isn't a marketing trick, but something that just… works.

Smart assistants, well-structured apps, experiments with the latest tech — whatever we make, we make sure it stands for something.

## 🧠 What we believe

> *"It's always easy to do something the way it's always been done.*
> *We'd rather do it the way it hasn't been done yet."*

- 🤖 **AI-first** — not because we have to, but because we can
- 🎨 **Creative by default** — we think in solutions, not in limitations
- 🌱 **Evergreen** — what we build keeps growing, long after we're done
- 🧪 **Playful** — if building isn't fun, we're doing something wrong
